Let's see how width works!

Width is not set on div. Takes all the available horizontal space.

Width is set to 200px.

Let's see how height works!

Height is not set on div. Height adapts to the content.
Having a second line increase the height.

Height is set to 200px.

This is a span. Height and width are not affected by the style.

Let's see how border works!

Border is not set on div. Box width is 0, so it does not show up.
Border property needs: width, style and type.

Border is set to 5px, solid, hotpink.

This is a span. Height and width are not affected by the style.

Let's see how Padding works!

Padding is not set on div. Padding is 0, so it does not add space between the height/width box and the border.

Padding is set to 25px. Space is created between the content and the border.

This is a span. Height and width are not affected by the style.

Let's see how Margin works!

Margin is not set on div. Margin is 0, so it does not add space between this element and the one surrounding it.

Margin is set to 25px. Space is created between the border of this element and the elements around.

This is a span. It is not affected by margin-top and margin-bottom.
